About this Demo

This Emergency Services demo is an offline-capable progressive web app created for demonstration purposes. It stores data locally in your browser and does not send personal data to any server.

Install

Use your browser install UI (Add to Home screen) or device install prompt to install the app. You can also use the manifest shortcuts on supported platforms.

Privacy

All profile data, contacts, and chat history remain in localStorage on your device. No server-side storage is used.
